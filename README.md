# Comtellion Services

A minimal web project containing an `index.html` file and supporting images. This repository represents the starting point for the Comtellion Services website or web application.

## Project Structure

```
index.html
images/
```

- **index.html**: The main HTML entry point for the site.
- **images/**: Directory for storing image assets used by the site.

## Getting Started

To view the project in a browser:

1. Clone the repository:
   ```bash
   git clone https://github.com/Suraj-Salihu/Comtellion-Services.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Comtellion-Services
   ```
3. Open `index.html` in a web browser.
